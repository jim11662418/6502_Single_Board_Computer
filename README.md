# 6502 Single Board Computer
A simple 6502 Single Board Computer based on [Grant Searle's 6502 Computer](http://searle.x10host.com/6502/Simple6502.html). The only real differences between Grant's design and this computer are the use of a GAL16V8 for 'glue' logic to reduce the chip count and the addition of a 6522 VIA for parallel I/O. Assemble the firmware with the [Macro Assembler AS](http://john.ccac.rwth-aachen.de:8000/as/)
